## Deferred Launch Inputs

The local product is runnable and verified by code-level gates, but a production-connected Amend deployment still needs external setup:

- Production web `VITE_CONVEX_URL` and `VITE_CONVEX_SITE_URL`.
- GitHub App or OAuth app slug/credentials, install flow, and live repository permissions.
- OpenAI or alternate AI provider key for legacy drafting paths and provider policy.
- Crof/Kimi `CROF_API_KEY`, `CROF_MODEL`, and `CROF_BASE_URL` for proactive agent runs.
- Resend API key plus verified sender/domain for real email delivery.
- Stripe credentials and verified billing webhooks before automatic production plan changes.
- A/AAAA/CNAME DNS records and host-based routing for custom portal/embed/API domains; apex
  ALIAS/ANAME/flattened CNAME records should resolve as A/AAAA answers.
- Registration/delegation of `amend.sh`; `whois -h whois.nic.sh amend.sh` currently returns
  `Domain not found`.
