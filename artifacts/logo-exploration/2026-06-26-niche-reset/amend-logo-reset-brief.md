# Amend logo reset

## Verdict

Use an A-splice mark: a hard geometric `A` with one hero-orange inserted patch.

This is the saner logo direction because it gives users brand recognition first. The orange patch carries "amend" as a small correction inside the letter, but the mark does not ask anyone to decode a workflow diagram.

Blunt critique: this is a good direction, not a finished brand system. The mark clears the loop/category trap and reads at small sizes. The wordmark still needs a real typography pass with outlined letters before it should replace production assets.

## Research takeaways

- Canny owns the obvious `C`/loop read in this category. Any orange loop for Amend looks like a Canny-adjacent repaint.
- Linear shows the right bar for devtool chrome: tiny, quiet, high-contrast, and not explanatory.
- Productboard uses a simple directional shard, not a literal roadmap or feedback-board icon.
- Dovetail and Fibery are more expressive, but Amend's own brand docs point colder and more source-native.

## Style DNA

- Geometry: hard-edged, compact, vector-native, no rounded app-tile softness.
- Color: near-black/soft-white plus one sampled hero ember, `#e19b40`.
- Metaphor: an amendment patch inserted into the brand letter.
- Small-size read: should still read as `A` at favicon and sidebar size.
- Mood: source-native developer tool, closer to Linear than product-feedback board software.

## Do not do

- No loop, circular route, horseshoe, magnet, refresh arrow, or `C` shape.
- No chat bubble, roadmap pin, checkbox, shield, document, file, folder, or fake code glyph.
- No multi-node workflow diagram.
- No gradient orb or decorative SaaS badge.
