# Amend Logo Exploration

## Recommended Idea

The strongest route is now the **hero ember C-loop**: a clean loop mark using the orange range from the current hero image, with one hot closing point. The earlier black return-route concepts were too chunky and read too much like a magnet/horseshoe; this direction is simpler, warmer, and much closer to the landing page.

## Why This Direction

- It matches the hero directly: deep ember orange, burnt gold, and a hot gold closing point.
- It keeps the existing "close the loop" idea without adding a complicated diagram.
- It works as a favicon because the silhouette is one C-loop plus one terminal.
- It feels more like Amend's landing page than the flat `#ef9836` UI accent.

## Generated Prompt Used

Premium logo/favicon source for Amend. Core promise: close the loop from customer request to shipped source evidence. Create one centered logo mark only, no wordmark. Concept: a compact near-circle return loop made from two hard-edged folded ribbon strokes, with a small warm amber signal dot completing the open end. Style: flat-dimensional vector mark, crisp geometric edges, near-black and white forms with one amber focal accent, developer infrastructure mood, readable at 32px, strong silhouette. Background/export: flat removable neon green chroma-key background for alpha removal, isolated source asset, no white or gray studio backdrop, no shadow plate. Avoid: generic arrows, checkmarks, shield, chat bubble, files, folders, code text, UI screenshots, orbiting nodes, 3D render, mascot, duplicate marks, wordmark, pale rim.

## Blunt Critique

The deep ember version is the strongest current direction. It still has a slight C-letter read, but that is much better than the previous magnet/route mark and fits the hero copy. The bright version is cleaner on white; the deep version is better on Amend's dark hero/nav.
